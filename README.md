# 阿里云函数计算官方文档

欢迎来到阿里云函数计算官方文档仓库。

本仓库用于组织和协作维护函数计算文档。文档内容基于公开的阿里云函数计算官方文档，并按产品领域重新归档，便于后续评审、贡献和长期维护。

## 文档范围

文档正文位于 `docs/`，按产品领域组织：

- 云沙箱
- 云函数
- 智能体 AgentRun
- 模型服务 FunModel
- 图像生成 FunArt

每个产品领域尽量保持统一的目录结构：

- 开始使用
- 产品动态
- 产品计费
- 功能说明
- 最佳实践
- 安全合规
- 开发参考
- 常见问题
- 服务支持

开发参考类内容由平台自动生成。本仓库只保留对应目录占位，不维护生成后的 API Reference 页面。

## 参与贡献

文档贡献不只包括直接修改文字，也包括反馈问题、补充缺失主题、指出失效链接、过期截图、错误示例或目录结构问题。

### 提交 Issue

欢迎内部和外部用户通过 Issue 反馈文档问题或改进建议。提交 Issue 时，建议包含：

- 受影响的文档路径或线上 URL；
- 发现的问题；
- 期望的修正方式或改进方向；
- 必要时附上截图或复现步骤。

维护者会定期处理 Issue，并将可执行事项转化为文档更新。

### 提交 Pull Request

欢迎开发者通过 Pull Request 贡献修订、补充和优化。提交 PR 前，请确认：

- 变更范围清晰；
- 内容准确，并有明确来源支撑；
- Markdown 引用的图片或媒体文件已同步提交；
- 不手工修改平台自动生成的开发参考内容；
- 在适用场景下保留原文档的结构和图文表达意图。

所有 PR 都需要维护者审核，并在 PTL Review 通过后合入。

## 仓库结构

- `docs/`：Markdown 格式的产品文档。
- `assets/`：文档引用的图片和媒体资源。
