# 使用约束

## **支持地域**

### **通过 E2B SDK 使用云沙箱（New）**

当前云沙箱 API/SDK 仅在以下地域可用：

- 华北2（北京）

### **通过沙箱函数使用云沙箱**

在以下地域的[函数计算控制台](https://fcnext.console.aliyun.com/)中，创建函数时可选择沙箱函数：

- 中国内地：华东1（杭州）、华东2（上海）、华北2（北京）、华南1（深圳）
- 中国香港及海外：中国香港、亚太东南1（新加坡）

### **深休眠**

深休眠功能目前处于内测阶段，仅在以下地域可用：

- 中国内地：华东1（杭州）、华东2（上海）、华北2（北京）、华南1（深圳）
- 中国香港及海外：中国香港、亚太东南1（新加坡）
