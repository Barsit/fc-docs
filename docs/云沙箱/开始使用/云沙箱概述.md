# 云沙箱概述

云沙箱（FC Sandbox）是一款专为 AI Agent、OpenClaw、RL 仿真等有状态长交互场景打造的高性能云沙箱，提供了一个安全隔离的云端沙箱环境来执行 AI 生成的代码。这种沙箱环境可以防止 AI Agent 访问或篡改系统之外的资源，确保其行为不会对系统造成损害。

## 产品优势

| **优势** | **说明** |
| --- | --- |
| **安全隔离的沙箱环境** | 沙箱提供基于 MicroVM 级别隔离的运行环境，提供计算、网络、存储端到端的安全隔离能力。 |
| **原生兼容 E2B，无缝迁移开箱即用** | 99% 兼容 E2B SDK，仅需修改一行 Endpoint 即可开箱即用。 |
| **原生支持会话亲和隔离** | 无需管理会话与实例映射，实现零运维。 |
| **高并发极致弹性** | 单 Region 百万级 Sandbox 并发和 10 万沙箱/分钟创建效率，支持缩容到 0。百毫秒冷启效率，资源池化预热可以做到毫秒级。自研调度+弹性网络，消除 ETCD 和 IP 资源瓶颈，毫秒级挂载效率。 |
| **休眠唤醒，极致降本** | 引入智能休眠能力，无请求自动浅休眠，有请求 1 ms 极速唤醒，典型场景降本 60%~80%。支持自动深休眠与 1 秒唤醒，保存内存状态+文件系统，实现断点续跑，做到效率、成本与体验的极致平衡。 |
| **极致冷启动性能** | 无需资源池预热，5000 并发 3 GB 异构镜像启动 P100 < 1 秒，性能行业领先。 |
| **动态挂载与高级特性** | ● 多源动态挂载：支持 NAS/OSS/PolarLakeBase/JuiceFS 动态挂载，实现云沙箱严格存储隔离。<br>● PolarLakeBase深度集成：支持海量数据向量化、上下文版本管理及百万级会话租户隔离与 Quota 管控。<br>● JuiceFS 企业级增强：解决高并发元数据与海量小文件瓶颈，支持模型缓存共享及跨云无缝访问。 |

## 适用场景

1. **有状态 Agent 长时交互**
  面向 AI Agent、OpenClaw、RL 仿真等有状态长交互场景，基于会话亲和性保持内存与文件上下文，消除频繁重启导致的状态丢失与延迟。
2. **安全代码执行与数据处理**
  在 MicroVM 隔离环境中远程执行 Python、Shell 等任意命令；内置 code-interpreter 模板，无缝对接 OSS、NAS 、PolarLakebase等企业主流存储，无需数据搬迁即可快速完成数据分析、文件处理及脚本执行，实现数据安全与开发效率的双重保障。
3. **浏览器自动化与复杂网页交互**
  CDP 协议连接远端 Chrome，精准操控动态渲染页面（SPA），稳定维持登录态与 Session，适配 RPA 流程自动化、前端 UI 测试及深度信息提取。
4. **海量仿真与大规模并行评估**
  面向 RL 任务及 Agent 能力评测，提供百万级并发支撑；凭借秒级启动 5000+ 异构 3GB 镜像实例的极致冷启效率，实现 5000 并发/秒的高吞吐创建，显著缩短大规模并行评估周期，极致优化算力成本。
5. **企业级开发运维一体化**
  基于自定义镜像预装依赖、工具及证书，构建标准化业务模板以确保环境一致性；利用 JuiceFS 企业版的跨云同步能力持久化代码与配置，实现开发状态的全球一致性与无缝保留，支持分布式团队高效协作；提供 CLI 工具链统一管理生命周期、日志及监控，实现研发运维高效闭环。
